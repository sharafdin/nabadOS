# nabadOS

nabadOS is a **64-bit CLI-based operating system** built from scratch for those who love the command line and the simplicity of a black screen. It features a custom bootloader written in **Assembly (NASM)** and a **Rust-based kernel**. The project follows the **Multiboot2 standard** and uses **GRUB** for booting. 

This is just the beginning—nabadOS is in its early stages, like an ancient relic in the digital age. But in the near future, we aim to expand it into a fully functional CLI-based OS capable of running on even the oldest devices. Maybe even **stone-age devices**—do they even have **vacuum tube computers**? Stay tuned for more!
